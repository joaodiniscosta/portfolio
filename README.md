# Personal Portfolio - João Costa

Welcome to the repository of my personal portfolio. This project serves as my digital business card, showcasing my professional journey, academic background, and technical projects.

## 🚀 Overview

This site is a dynamic reflection of my trajectory as a Software Engineer and Team Leader, built to provide a clear and professional overview of my career and skills.

- **Profile**: A brief professional summary of my evolution from full-stack development to technical leadership.
- **Experience**: Detailed professional history, highlighting my roles at Deloitte and Samsys.
- **Education**: Information on my academic path, including my Bachelor's and current Master's degree in Cybersecurity.
- **Projects**: A showcase of my personal and professional work, including key initiatives.

## 🛠️ Tech Stack

This project is built using:

- **Frontend**: React
- **Styling**: React-Bootstrap
- **Animations**: Framer Motion
- **Deployment**: GitHub

## 📂 Structure

- `src/components/`: Modular UI components (Experience, Education, etc.).
- `src/assets/`: Media and static files.
- `public/`: Public assets and manifest.
